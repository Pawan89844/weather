## WEATHER APP

Weather app scraps data from accuweather to display the weather report.

## Features

1. Live Weather
2. Fetches Location
3. Provides weather report and temperature.

## Dart Features

1. OOPs (Object oriented)
2. REG EXP
3. String operations

## Packages

1. Provider
2. HTTP
3. HTML
4. SVH
5. GEO-Locator
6. GEO-Coding
